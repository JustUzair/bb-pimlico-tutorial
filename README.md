# Pimlico x BuildBear Tutorial

To set up the tutorial, clone this repository, run install the dependencies, and run `npm start`!

```bash
npm install
npm start
```

# How to use smart account actions

- Use the flags:
  - `-- --send-transaction`
  - `-- --send-userOp`
  - `-- --estimate-userOp-gas`
  - `-- --get-userOp-receipt`
